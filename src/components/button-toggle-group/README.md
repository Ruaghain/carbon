## Button Toggle Group.

### How to use a Button Toggle Group in a component:

* In your file:

```javascript
import ButtonToggleGroup from 'carbon-react/lib/components/button-toggle-group';
```

* To render the Button Toggle Group:

```javascript
<ButtonToggleGroup value={ 'foo' } validations={ [] }>
  <ButtonToggle />
</ButtonToggleGroup>
```
